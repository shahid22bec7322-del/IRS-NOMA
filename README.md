# IRS-Assisted NOMA for 5G/6G Wireless Networks

## Introduction
This project focuses on combining **Intelligent Reflecting Surface (IRS)** with **Non-Orthogonal Multiple Access (NOMA)** to improve wireless communication performance in 5G/6G networks.  

- **NOMA:** Allows multiple users to share the same spectrum efficiently.  
- **IRS:** Intelligently reflects signals to improve coverage, reduce outage probability, and enhance signal reliability in fading environments.  

The project is divided into **scenario-based studies** to achieve expected performance outcomes.

---

## Scenarios

### Scenario 1: IRS-NOMA vs Conventional NOMA under Passive IRS, Perfect SIC
- Designed and simulated IRS-assisted NOMA system with **Passive IRS** and **Perfect SIC**.  
- Achieved measurable gains in **sum-rate capacity** over conventional NOMA across **Rayleigh and Rician fading channels**.  

### Scenario 2: IRS-NOMA vs Conventional NOMA under Active IRS, Imperfect SIC
- Designed and simulated IRS-assisted NOMA system with **Active IRS** and **Imperfect SIC**.  
- Achieved improved sum-rate performance over conventional NOMA.  

### Scenario 3: Energy Efficiency (EE) Comparison
- Compared **Passive IRS-NOMA vs Active IRS-NOMA**.  
- Analyzed key system metrics:  
  - **Bit Error Rate (BER)**  
  - **Outage Probability**  
  - **Sum-Rate Capacity**  
  - **Energy Efficiency**  
- Evaluated under varying **SNR** and **channel conditions**.  

### Scenario 4: IRS-NOMA under Eta-Mu Fading
- Extended simulation to **Eta-Mu generalized fading** under **Active IRS with Imperfect SIC**.  
- Demonstrated broader fading model applicability beyond standard Rayleigh/Rician models.  

### Scenario 5: Why NOMA if OFDM is already available?
- Comparative study: **IRS-assisted NOMA vs IRS-assisted OFDMA**  
- Active IRS with Imperfect SIC configuration.  
- Provided **analytical justification** for NOMA's spectral efficiency advantage.  

### Scenario 6: IRS-NOMA Single-Cell vs Multi-Cell
- Modeled **multi-cell interference** using **Stochastic Geometry (Poisson Point Process – PPP)**.  
- Proposed a **Distributed Algorithm (DA)** for inter-cell interference mitigation in IRS-NOMA deployments.  

---

## Key Achievements
- Demonstrated **sum-rate gains** with IRS-assisted NOMA over conventional NOMA.  
- Validated system performance in **different fading channels** including Rayleigh, Rician, and Eta-Mu.  
- Conducted **energy efficiency analysis** for passive vs active IRS configurations.  
- Justified **NOMA's advantage over OFDMA** in IRS-assisted scenarios.  
- Proposed a **distributed solution** for multi-cell interference in practical deployments.
